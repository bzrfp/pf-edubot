# Projeto final - Treinamento Edubot 2026.1
  
  O objetivo deste projeto é programar o robô Sparki tal que ele seja capaz de realizar o labirinto proposto
na figura abaixo.   

  Durante o percurso, sempre que o Sparki encontrar um caminho sem saída, ele deverá verificar a presença
de um objeto utilizando seus sensores e seu mecanismo de garra. Caso um objeto seja detectado e capturado, o
robô deverá interromper imediatamente a exploração e retornar ao ponto inicial do labirinto. Caso contrário,
deverá continuar a navegação até localizar o objeto.
